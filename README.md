### Prerequisites

[**Node.js**](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

### Installation
Clone the repository:

Then install the project dependency

```bash
cd eluviosde_oa
npm install
```

### Run

```bash
node index.js
```
